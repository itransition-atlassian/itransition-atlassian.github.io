User guide Part III. Telegram Chats Page

After the bot is added, you can create a task chat or a project chat on the chat's page.

The chat's page is available after installing the plugin and completing admin authorization the link is located on the project's sidebar:

On the chat's page, you can see all the project chats if you are a project administrator, or chats where you are the owner or member.

On this page, you can create a new chat or edit an existing one.

You can use the following parameters:

  1. Type - chat type. It can have a value: Issue, Project.
  2. Issue - contains the issue key for the Issue chats, it is non-editable for Project chat.
  3. Auto post comments - this parameter determines whether all messages in the Telegram chat written by participants will be published as comments to the issue.
  4. Notification - this parameter determines whether notifications will be sent to the Telegram chat.
    Note: if you do not specify it, you won't receive notifications in Telegram, also, this parameter could not be changed after chat creation
  5. Issue events - a list of events for which notifications will come to the chat.
    6. For Issue chats:
    1. Issue Resolved
    2. Issue Closed
    3. Issue Assigned
    4. Issue Commented
    5. Issue Comment Edited
    6. Issue Reopened
    7. Issue Progress Started
    8. Issue Generic Event
    For Project chats:
    1. Issue Created
    2. Issue Deleted
    3. Issue Resolved
    4. Issue Closed
    5. Issue Assigned
    6. Issue Commented
    7. Issue Comment Edited
    8. Issue Reopened
    9. Issue Progress Started
    10. Issue Generic Event
  6. Owner - a non-editable parameter, automatically sets the current user. The owner in the Telegram chat will be considered the chat owner and will have the chat administrator rights.
  7. Users - a list of chat members who are in the chat and have the rights of regular users.
  8. Administrators - a list of chat members who are in the chat and have the rights of Telegram chat administrators.
  9. Action - for users who are not the owners of the chat, a Leave button is displayed in this column. Upon clicking, the current user exits the chat.

Note: Only authorized Telegram users that have access to the project are provided in chat member lists (Users, Administrators).

Product Overview

In this quick overview we will go over basic functionality of plugin.

  1. What features does the plugin provide?
    • Create Telegram chats directly from JIRA issue view or project screen.
    • Add to Telegram chat new members, remove existing chat participants, manage admin privileges.
    • Receive notifications about issue status from Telegram bot.
    • Post messages from Telegram chat to linked issue in Jira.
  2. What do you need to do?
    • To use all provided features, you need to create Telegram account (if you don't have one).
